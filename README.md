## Tips

## Features

- Authentication & Authorization
- Admin & User panel
- Add & remove category (admin rule)
- Add & remove post
- Search posts
- Filter posts
- Post details page
- SPA app
- Login / Logout

- React JS

- Tailwind CSS
- Headless UI
- React router dom (v6)
- React hot toast
- React loading
